# PROJECT
Markus AI Sidekick Website Assistant
Personal Project

# PROJECT DETAILS
## Date
Jan 2026 (last updated)

## Summary
Developed an AI sidekick running locally on an open-source model model for my personal website. Markus is tasked to sing my praises and help users navigate the site and its contents. He has grounded details on me and my projects

## Details

- Markus runs on a mac mini bought off facebook marketplace using a small open-source model (free!)
- Markus has been fed documents to ground him on his knowledge of his creator (me, Brandon) and my projects.
- Markus' personality has been engineered to entertain website goers and support his creator.

# PROJECT TECH STACK
- TypeScript
- Ollama

# INTERESTING COMPONENTS
## Local Inference
- Serving a local model to the internet via Cloudflare tunnel for conversational AI functionality without API costs
- Working to maximise performance of smaller open-source models while prioritising latency

## Managed Context
- Context is being managed manually at backend.
- Experimenting with different techniques for effficient context engineering (especially with limited context window)

## Document Grounding
- To improve open-source model performance, semantic grounding on source documents is being used.
